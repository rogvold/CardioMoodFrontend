Использование сайта CardioMood

Сайт kolomna.cardiomood.com предоставляет интерфейс для непрерывного мониторинга состояния спортсменов во время тренировки.

  • Вход в систему и мониторинг
  • Изменение названия измерения на сайте

Использование мобильного приложения CardioMood

Приложение для Android необходимо для сбора данных в процессе тренировки. Данные, полученные от пульсометра Polar H7, передаются на мобильный телефон через беспроводной интерфейс Bluetooth. При наличи интернет соединения помимо сохранения в телефоне, данные отправляются на сервер, позволяя тренеру удаленно вести наблюдения за показателями в реальном времени.

  • Требования к оборудованию
  • Создание аккаунта и вход
  • Настройки программы
  • Установка соединения с датчиком Polar H7
  • Работа программы во время тренировки
  • Просмотр отчетов и истории тренировок
  • Синхронизация с сервером
  • Решение проблем
